# 📊 HR Analytics Dashboard (Power BI)

## 🔍 Project Overview

This project focuses on analyzing HR data to uncover insights related to employee distribution, salary trends, workforce demographics, and organizational growth.

The goal is to help HR teams make **data-driven decisions** around hiring, compensation, and workforce planning.

---

## 🎯 Business Questions Solved

This dashboard answers key HR questions:

* How many employees are there in each job role?
* What is the gender distribution across the organization?
* What is the age distribution of employees?
* Which job roles offer higher salaries?
* Who are the top earners in each job category?
* How does qualification impact salary?
* How has employee count grown over time?
* What is the leave balance distribution?
* Can we dynamically filter employees (e.g., by name)?

---

## 📊 Dashboard Features

* Interactive filters and slicers
* Dynamic employee search (starting letter filter)
* Salary comparison across roles
* Time-based workforce trend analysis
* Clean and user-friendly UI

---

## 🛠 Tools & Technologies Used

* Power BI
* Data Cleaning & Transformation (Power Query)
* Data Visualization
* Basic Data Modeling

---

## 📈 Key Insights

* Certain job roles significantly outperform others in salary distribution
* Workforce growth shows consistent upward trend over time
* Higher qualifications tend to correlate with higher salaries
* Gender distribution is uneven across some departments
* Leave balance varies widely, indicating potential policy gaps
